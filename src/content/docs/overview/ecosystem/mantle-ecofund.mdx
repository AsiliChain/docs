# Mantle EcoFund

The Mantle EcoFund is a $200 million strategic fund supporting projects building on Mantle Network. It is AsiliChain's primary Phase 1 and Phase 2 funding target.

## Why AsiliChain Fits the EcoFund Mandate

| EcoFund priority | AsiliChain alignment |
|-----------------|---------------------|
| RWA tokenization | BatchToken is an agricultural commodity RWA — the next frontier after US Treasuries |
| Emerging market DeFi | Uganda + East Africa. The segment where DeFi has the highest real-world impact per dollar deployed |
| Institutional credibility | EUDR compliance, government data integration, UMRA-licensed MFI partners |
| Non-speculative token utility | No native speculative token. USDC-denominated. Protocol fees fund treasury. |
| TVL growth for Mantle | LendingVault USDC pool grows Mantle TVL. DeFi composability with Aave V3 on Mantle. |

## Funding Strategy

### Phase 1 Target: $50,000–$100,000

**Use:** Seed the LendingVault for 200-farmer pilot. Fund 3 months of development costs. Pay Kotani Pay and MAAIF integration costs.

**Ask:** EcoFund grant (non-dilutive). No token required. Mantle branding on documentation and dashboard.

**Deliverable:** 6-month pilot report with real repayment data from verified Uganda coffee farmers, publishable as an EcoFund impact case study.

### Phase 2 Target: $500,000–$1M

**Use:** Scale to 2,000 farmers across 5 cooperatives. Fund MFI onboarding (UGAFODE, ENCOT). Cover EU TRACES integration costs.

**Ask:** EcoFund strategic investment or ecosystem grant. Mantle validator node consideration.

**Deliverable:** Operational protocol with $1M+ LendingVault, 8–10% MFI yield, verified auto-repayment track record.

## Application Process

The EcoFund application requires:

1. **Technical review** — Mantle core team reviews contract architecture
2. **Impact assessment** — documented farmer outcomes methodology
3. **Legal review** — Mantle TaaS framework alignment verification
4. **Partnership agreement** — co-marketing, case study rights, technical support
