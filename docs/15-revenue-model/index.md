title: 15. Revenue Model
sidebar_position: 16
---

AsiliChain is a protocol, not a middleman. Revenue is earned from value added to the system. The protocol deliberately carries no token at this stage. Token economics introduce regulatory risk and community management overhead that is premature before the core protocol has demonstrated product-market fit.

| Revenue Stream | Description | Active From |
| --- | --- | --- |
| Interest rate spread | Spread between borrower APR and MFI liquidity pool rate. Set by governance. Disclosed to all participants. | Phase 2 |
| Protocol fee (0.5%) | Fee on all BatchToken transfers. 70% treasury, 20% cooperative pool, 10% operations. Multisig-governed. | Phase 3 |
| Data licensing API | Paid API for exporters, UCDA, auditors. Per-query or monthly subscription. | Phase 3 |
| EUDR DDS generation fee | Per-statement fee for auto-generated DDS and IPFS pinning. Priced below cost of manual compliance documentation. | Phase 3 |
