Brand Kit

Brand system

AsiliChain Brand Kit

A disciplined visual identity built around forest green for trust, harvest amber for financial warmth, chain teal for traceability, and soil neutrals for clarity.

Forest GreenHarvest AmberChain TealSoil Neutrals

Logo system​

Primary logo

Use this lockup in the navbar, hero sections, social cards, and public-facing materials.

Icon mark

Use the icon mark for tight spaces, avatars, and small-format applications.

Wordmark studies​

Wordmark study A

Higher placement and tighter spacing for a more compact read against the icon.

Wordmark study B

Current balanced baseline with the most neutral ascender treatment.

Wordmark study C

Lower placement and slightly looser tracking for a calmer lockup.

Clear space and usage​

  • Keep a generous amount of breathing room around the logo.
  • Do not stretch, recolor, outline, or add effects.
  • Prefer the full logo in public-facing contexts.
  • Use the icon alone only when space is limited.

Typography​

  • Headings should be strong, modern, and readable.
  • Body text should remain neutral and highly legible.
  • Avoid decorative typefaces in core protocol documentation.

Application guidance​

  • Use forest green for primary navigation and trust cues.
  • Use harvest amber sparingly for financial or cautionary emphasis.
  • Use chain teal for technical flows, traceability, and data states.
  • Keep layouts warm and legible with soil neutrals.
